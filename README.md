## ✨ What is Mimosa Desktop?

Mimosa Desktop is a desktop messaging client compatible with the [Mimosa Server](https://github.com/Mimosa/Mimosa-server). Based on Telegram Desktop, it provides a familiar interface for connecting to your own self-hosted messaging server.

## 🔗 Compatibility

This client is designed to work with the [Mimosa Server](https://github.com/Mimosa/Mimosa-server). Simply configure the client to connect to your Mimosa server instance and start messaging!

## ⚙️ Server Configuration

To change the server address, replace all instances of `XXX.XXX.XXX.XXX` in the code with your server's IP address or domain name.
